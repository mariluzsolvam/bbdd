La estructura del proyecto la podéis ver en el Aula Virtual, en la asignatura de Base de datos. Es un buen comienzo hacer esta estructura para tener todos los archivos del proyecto bien organizados.

Base de datos BREAINS

El diseño de la base de datos lo podéis ver en el Aula Virtual, en la asignatura de Base de datos. La base de datos está formada por 3 tablas, de la cuales dos están relacionadas y la otra no.

Crear la base de datos e insertar datos de Internet. Al mismo tiempo que recopiláis información, descargar una imagen relacionada con cada artículo o cada detalle de la base de datos.

En la base de datos, en el campo "imagen" de cada tabla, solo se almacena el nombre de la imagen. La imagen en realidad está en una carpeta y esta se mostrará cuando indiques la ruta completa con la etiqueta img de HTML. (Lo veremos más adelante)

Redimensionar imágenes para el proyecto

Una forma de adaptar las imágenes a los div de la plantilla html, es utilizar la aplicación Gimp para escalar las imágenes.

También podrías utilizar las propiedades width y heigh de html para ponerlas al tamaño indicado en el proyecto.

Crear conexión php.

Abre un editor HTML, por ejemplo Brackets, Geany...

Escribe el código php de la conexion y guárdalo como conexion.php. Después, guárdalo o súbelo al servidor a la carpeta php del proyecto.

A continuación, incluye el archivo conexion.php en index.php de la carpeta php.
